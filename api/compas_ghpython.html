compas_ghpython

This package contains utilities and helpers for working with COMPAS in Grasshopper.

drawing

xdraw_frame(frame)

Draw frame.

xdraw_points(points)

Draw points.

xdraw_lines(lines)

Draw lines.

xdraw_polylines(polylines)

Draw polylines.

xdraw_faces(faces[, srf, u, v, trim, …])

Draw polygonal faces as Breps.

xdraw_cylinders(cylinders[, cap])

xdraw_pipes(pipes[, cap, fit])

xdraw_spheres(spheres)

xdraw_mesh(vertices, faces[, name, …])

Draw mesh in Grasshopper.

xdraw_network(network)

Draw a network in Grasshopper.

mesh_draw(mesh[, show_faces, show_vertices, …])

Draw a mesh object in Grasshopper.

sets

list_to_ghtree(alist[, none_and_holes, …])

Transforms nestings of lists or tuples to a Grasshopper DataTree.

ghtree_to_list(atree)

Returns a list representation of a Grasshopper DataTree

timers

update_component(ghenv, delay)

Schedule an update of the Grasshopper component.

utilities

unload_modules(top_level_module_name)

Unloads all modules named starting with the specified string.
