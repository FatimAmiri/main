Mesh.vertex_neighborhood

Mesh.vertex_neighborhood(key, ring=1)

Return the vertices in the neighborhood of a vertex.

Parameters

  • key (hashable) – The identifier of the vertex.

  • ring (int, optional) – The number of neighborhood rings to include. Default is 1.

Returns

list – The vertices in the neighborhood.

Note

The vertices in the neighborhood are unordered.

Example

import compas
from compas.datastructures import Mesh
from compas_plotters import MeshPlotter

mesh = Mesh.from_obj(compas.get('faces.obj'))

key = 17
nbrs = mesh.vertex_neighborhood(key, ring=2)

plotter = MeshPlotter(mesh)

color = {nbr: '#cccccc' for nbr in nbrs}
color[key] = '#ff0000'

text = {nbr: str(index) for index, nbr in enumerate(nbrs)}
text[key] = str(key)

plotter.draw_vertices(text=text, facecolor=color)
plotter.draw_faces()
plotter.draw_edges()

plotter.show()
