Circle.transformed

Circle.transformed(transformation)

Returns a transformed copy of the current circle.

Parameters

transformation (Transformation) – The transformation used to transform the circle.

Returns

circle – The transformed circle.

Examples

>>> from compas.geometry import Frame
>>> from compas.geometry import Transformation
>>> from compas.geometry import Plane
>>> from compas.geometry import Circle
>>> circle = Circle(Plane.worldXY(), 5)
>>> frame = Frame([1, 1, 1], [0.68, 0.68, 0.27], [-0.67, 0.73, -0.15])
>>> T = Transformation.from_frame(frame)
>>> circle_transformed = circle.transformed(T)
