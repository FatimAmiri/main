MeshArtist

class compas_ghpython.artists.MeshArtist(mesh)[source]

Bases: compas_ghpython.artists.mixins.faceartist.FaceArtist, compas_ghpython.artists.mixins.edgeartist.EdgeArtist, compas_ghpython.artists.mixins.vertexartist.VertexArtist

A mesh artist defines functionality for visualising COMPAS meshes in GhPython.

Parameters

mesh (compas.datastructures.Mesh) – A COMPAS mesh.

Attributes

defaults (dict) – Default settings for color, scale, tolerance, …

Examples

import compas
from compas.datastructures import Mesh
from compas_ghpython.artists import MeshArtist

mesh = Mesh.from_obj(compas.get('faces.obj'))

artist = MeshArtist(mesh)
artist.draw_faces(join_faces=True)
artist.draw_vertices(color={key: '#ff0000' for key in mesh.vertices_on_boundary()})
artist.draw_edges()
