VolMeshArtist

class compas_rhino.artists.VolMeshArtist(volmesh, layer=None)[source]

Bases: compas_rhino.artists.mixins.faceartist.FaceArtist, compas_rhino.artists.mixins.edgeartist.EdgeArtist, compas_rhino.artists.mixins.vertexartist.VertexArtist

A volmesh artist defines functionality for visualising COMPAS volmeshes in Rhino.

Parameters

  • volmesh (compas.datastructures.VolMesh) – A COMPAS volmesh.

  • layer (str, optional) – The name of the layer that will contain the volmesh.

defaults

dict – Default settings for color, scale, tolerance, …
