VolMeshArtist

class compas_rhino.artists.VolMeshArtist(volmesh, layer=None)

Bases: compas_rhino.artists.mixins.FaceArtist, compas_rhino.artists.mixins.EdgeArtist, compas_rhino.artists.mixins.VertexArtist, compas_rhino.artists.Artist

A volmesh artist defines functionality for visualising COMPAS volmeshes in Rhino.

Parameters

  • volmesh (compas.datastructures.VolMesh) – A COMPAS volmesh.

  • layer (str, optional) – The name of the layer that will contain the volmesh.

Attributes

defaults (dict) – Default settings for color, scale, tolerance, …
