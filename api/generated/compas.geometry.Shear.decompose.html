Shear.decompose

Shear.decompose()

Decomposes the Transformation into Scale, Shear, Rotation, Translation and Perspective.

Example

>>> trans1 = [1, 2, 3]
>>> angle1 = [-2.142, 1.141, -0.142]
>>> scale1 = [0.123, 2, 0.5]
>>> T1 = Translation(trans1)
>>> R1 = Rotation.from_euler_angles(angle1)
>>> S1 = Scale(scale1)
>>> M = (T1 * R1) * S1
>>> Sc, Sh, R, T, P = M.decompose()
>>> S1 == Sc
True
>>> R1 == R
True
>>> T1 == T
True
