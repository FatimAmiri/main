Vector.length_vectors

static Vector.length_vectors(vectors)[source]

Compute the length of multiple vectors.

Parameters

vectors (list) – A list of vectors.

Returns

list – A list of lengths.

Examples

>>> Vector.length_vectors([[1.0, 0.0, 0.0], [2.0, 0.0, 0.0]])
[1.0, 2.0]
