RhinoCurve

class compas_rhino.geometry.RhinoCurve(guid)

Bases: compas_rhino.geometry.RhinoGeometry

Wrapper for Rhino curves.

Parameters

guid (str or System.Guid) – The GUID of the Rhino curve object.

Examples

import compas_rhino

from compas_rhino.geometry import RhinoCurve

curve = RhinoCurve(guid)

points = curve.divide(10)
tangents = curve.tangents(points)

for point, vector in zip(points, tangents):
    print point, tangent
