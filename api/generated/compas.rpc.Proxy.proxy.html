Proxy.proxy

Proxy.proxy(*args, **kwargs)

Callable replacement for the requested functionality.

Parameters

  • args (list) – Positional arguments to be passed to the remote function.

  • kwargs (dict) – Named arguments to be passed to the remote function.

Returns

object – The result returned by the remote function.

Warning

The args and kwargs have to be JSON-serialisable. This means that, currently, only native Python objects are supported. The returned results will also always be in the form of built-in Python objects.
