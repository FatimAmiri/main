Getting started

Installation

The recommended way to install COMPAS is to use Anaconda/conda:

$ conda config --add channels conda-forge
$ conda install COMPAS

But it can also be installed using pip:

$ pip install COMPAS

To verify your setup, start Python from the command line and run the following:

>>> import compas
>>> import compas_rhino
>>> import compas_blender
>>> import compas_ghpython

Optionally, you can also install from source. Check the Developer Guide for more info. To install the bleeding edge version directly from the GitHub repo

$ pip install git+https://github.com/compas-dev/compas.git
