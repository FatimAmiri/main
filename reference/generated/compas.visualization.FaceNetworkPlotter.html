FaceNetworkPlotter

class compas.visualization.FaceNetworkPlotter(network, **kwargs)[source]

Bases: compas.visualization.plotters.networkplotter.NetworkPlotter

Definition of a plotter object based on matplotlib for compas FaceNetworks.

Parameters:network (object) – The FaceNetwork to plot.
title

str – Title of the plot.

facecollection

object – The matplotlib collection for the mesh faces.

defaults

dict – Dictionary containing default attributes for vertices and edges.
