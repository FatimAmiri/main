print_profile

compas.utilities.print_profile(func)[source]

Decorate a function with automatic profile printing.

Parameters:func (function) – The function to decorate.
Returns:function – The decorated function.

Examples

@print_profile
def f(n):
    """Sum up all integers below n."""
    return sum(for i in range(n))

print(f(100))
print(f.__doc__)
print(f.__name__)
