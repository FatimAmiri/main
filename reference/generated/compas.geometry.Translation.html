Translation

class compas.geometry.Translation(translation)[source]

Bases: compas.geometry.xforms.transformation.Transformation

Creates a translation transformation.

Parameters

translation (list of float) – a list of 3 numbers defining the translation in x, y, and z.

Example

>>> T = Translation([1, 2, 3])
