KDTree.nearest_neighbour

KDTree.nearest_neighbour(point, exclude=None)[source]

Find the nearest neighbour to a given point, excluding neighbours that have already been found.

Parameters

  • point (list) – XYZ coordinates of the base point.

  • exclude (set, optional) – A set of points to exclude from the search. Defaults to an empty set.

Returns

list – XYZ coordinates of the nearest neighbour. Label of the nearest neighbour. Distance to the base point.
