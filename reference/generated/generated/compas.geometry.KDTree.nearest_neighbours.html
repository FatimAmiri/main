KDTree.nearest_neighbours

KDTree.nearest_neighbours(point, number, distance_sort=False)[source]

Find the N nearest neighbours to a given point.

Parameters

  • point (list) – XYZ coordinates of the bbase point.

  • number (int) – The number of nearest neighbours.

  • distance_sort (bool, optional) – Sort the nearest neighbours by distance to the base point. Default is False.

Returns

list – A list of N nearest neighbours.
