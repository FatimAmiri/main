Scale.inverse

Scale.inverse()

Returns the inverse transformation.

Example

>>> from compas.geometry import Frame
>>> f = Frame([1, 1, 1], [0.68, 0.68, 0.27], [-0.67, 0.73, -0.15])
>>> T = Transformation.from_frame(f)
>>> I = Transformation() # identity matrix
>>> I == T * T.inverse()
True
