Projection.orthogonal

classmethod Projection.orthogonal(point, normal)[source]

Returns an orthogonal Projection to project onto a plane defined by point and normal.

Parameters

  • point (list of float)

  • normal (list of float)

Example

>>> point = [0, 0, 0]
>>> normal = [0, 0, 1]
>>> P = Projection.orthogonal(point, normal)
