network.algorithms

network_count_crossings(network) Count the number of crossings (pairs of crossing edges) in the network.
network_dual(network[, cls]) Construct the dual of a network.
network_embed_in_plane(network[, fix, ...]) Embed the network in the plane.

Mesh

Package for working with mesh objects.

Mesh() Definition of a mesh.

mesh.operations

mesh_collapse_edge(mesh, u, v[, t]) Collapse an edge to its first or second vertex, or to an intermediate point.
mesh_split_edge(mesh, u, v[, t, allow_boundary]) Split and edge by inserting a vertex along its length.
mesh_split_face(mesh, fkey, u, v) Split a face by inserting an edge between two specified vertices.
mesh_unweld_vertices(mesh, fkey[, where]) Unweld a face of the mesh.

The following operations are designed for triangle meshes.

trimesh_collapse_edge(mesh, u, v[, t, ...]) Collapse an edge to its first or second vertex, or to an intermediate point.
trimesh_split_edge(mesh, u, v[, t, ...]) Split an edge of a triangle mesh.
trimesh_swap_edge(mesh, u, v[, allow_boundary]) Replace an edge of the mesh by an edge connecting the opposite vertices of the adjacent faces.

mesh.algorithms

mesh_delaunay_from_points(points[, polygon, ...]) Computes the delaunay triangulation for a list of points.
mesh_dual(mesh[, cls]) Construct the dual of a mesh.
mesh_flip_cycles(mesh) Flip the cycle directions of all faces.
mesh_subdivide(mesh[, scheme]) Subdivide the input mesh.
mesh_subdivide_tri(mesh[, k]) Subdivide a mesh using simple insertion of vertices.
mesh_subdivide_catmullclark(mesh[, k, fixed]) Subdivide a mesh using the Catmull-Clark algorithm.
mesh_subdivide_doosabin(mesh[, k, fixed])
mesh_unify_cycles(mesh[, root]) Unify the cycle directions of all faces.
mesh_voronoi_from_points(points[, boundary, ...]) Construct the Voronoi dual of the triangulation of a set of points.
mesh_cull_duplicate_vertices(mesh[, precision]) Cull all duplicate vertices of a mesh and sanitize affected faces.

The following operations are designed for triangle meshes.

trimesh_remesh(mesh, target[, kmax, tol, ...]) Remesh until all edges have a specified target length.
trimesh_subdivide_loop(mesh[, k, fixed]) Subdivide a triangle mesh using the Loop algorithm.
