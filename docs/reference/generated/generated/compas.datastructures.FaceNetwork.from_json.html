FaceNetwork.from_json(filepath)

Construct a datastructure from structured data contained in a json file.

Parameters:filepath (str) – The path to the json file.
Returns:object – An object of the type of cls.

Note

This constructor method is meant to be used in conjuction with the corresponding to_json method.
