Mesh.set_faces_attribute

Mesh.set_faces_attribute(name, value, fkeys=None)

Set one attribute of multiple faces.

Parameters

  • name (str) – The name of the attribute.

  • value (object) – The value of the attribute.

  • keys (iterable (None)) – A list of face identifiers. Defaults to all faces.
