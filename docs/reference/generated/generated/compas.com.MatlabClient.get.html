MatlabClient.get

MatlabClient.get(name)[source]

Get the value of a variable in the workspace.

Parameters:name (str) – The name of the variable.
Returns:str, int, float, list – The value of the variable.

Examples

>>> m = MatlabClient(verbose=True)

>>> m.get('A')
[[1, 0, 1, 3], [2, 3, 4, 7], [-1, -3, -3, -4]]
