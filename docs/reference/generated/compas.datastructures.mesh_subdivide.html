mesh_subdivide

compas.datastructures.mesh_subdivide(mesh, scheme='tri', **options)[source]

Subdivide the input mesh.

Parameters:
  • mesh (Mesh) – A mesh object.
  • scheme (str) – Optional. The scheme according to which the mesh should be subdivided. Defult is ‘tri’. Supported values are:
    • tri: insertion subdivision.
    • corner: corner cutting.
    • catmullclark : catmull-clark subdivision.
    • doosabin : doo-sabin subdivision.
  • options (kwargs) – Optional additional keyword arguments.
Returns:

Mesh – The subdivided mesh.
Raises:
  • NotImplementedError
  • MeshError
