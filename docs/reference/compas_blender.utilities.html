utilities

This package contains many convenience functions for working and interacting with Blender.

document

layers

clear_layer(layer) Deletes objects in given layer.
clear_layers(layers) Deletes objects in given layers.
layer_mask(layer) Creates a boolean layer mask.

delete_objects(objects) Delete a list of objects.

get_objects_name(objects) Retrieves the names of objects.

get_objects_location(objects) Retrieves the location of objects.
get_points() Retrieves all Empty objects in scene.
get_curves() Retrieves all Curve objects in scene.
get_meshes() Retrieves all Mesh objects in scene.

set_objects_location(objects, locations) Set the location of objects.
set_object_rotation(object, rotation) Set the rotation of an object.
set_objects_rotation(objects, rotations) Set the rotation of objects.
set_object_scale(object, scale) Set the scale of an object.
set_objects_scale(objects, scales) Set the scale of objects.
join_objects(objects) Join a list of objects.
select_point() Select point (empty) object.
select_points() Select points (empty) objects.
select_curve() Select curve object.
select_curves() Select curve objects.
select_mesh() Select Blender mesh object.
select_meshes() Select Blender mesh objects.
select_object(object) Select specific object.
select_objects(objects) Select specific objects.
select_all_objects() Select all objects.
deselect_object(object) De-select a specific object.
deselect_objects(objects) De-select specific objects.
deselect_all_objects() De-select all objects.
hide_object(object) Hide specific object.
hide_objects(objects) Hide specific objects.
show_object(object) Show specific object.
show_objects(objects) Show specific objects.
